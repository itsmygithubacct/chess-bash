# Security policy

Chess Bash has no network service and stores no user data. Its external process
surface is limited to the optional UCI engine selected by `STOCKFISH` and the
local audio sink commands documented in the README.

For a potentially sensitive vulnerability, use GitHub's private vulnerability
reporting for this repository. Ordinary crashes and compatibility bugs can be
reported through the public issue tracker.

Security fixes target the latest release. Include a minimal reproducer, affected
version, platform, and expected impact; do not include credentials or personal
data in a report.
